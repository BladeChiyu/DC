# FAQ

Q: Do I need to apply the patch every update?
A: Yes for every major update (WB/Ragna/ND) and you have to get the a new patch from here
Q: How do I apply the patch?
A: Steps to apply patch:
  1. Download one of the patches
  2. Rename the file to “locale.pck”
  3. Move the file into your games files folder
    a. On android move it to Android>data>com.stairs.destinychild>files
      b. On PC (DMM) move it to the “DESTINY_CHILD_for_DMM” folder
        i. You can get to that folder by clicking on the (i) icon on the game launcher and the folder icon on the bottom right in the pop up
